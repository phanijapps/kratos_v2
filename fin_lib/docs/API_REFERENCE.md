# fin_lib API Reference

## Overview

fin_lib is a professional financial analysis library designed for AI agents and quantitative analysis. This documentation is organized by area of expertise for easy navigation.

**Live Samples:** Real responses captured on 2025-11-08 are stored in `fin_lib/docs/api/sample_outputs.json`. Each API page shows a trimmed snippet; consult the JSON file for the complete data.

## API Documentation by Area

### [Base API](api/base.md)
Core stock market functions, installation, and quick start guide.
- Stock quotes and historical data
- Symbol search and market status
- Basic data retrieval functions

### [Technical Analysis](api/technical.md)
Technical indicators and chart analysis functions.
- RSI, MACD, Bollinger Bands
- Comprehensive technical analysis
- Custom indicator calculations

### [Fundamental Analysis](api/fundamentals.md)
Company data, financial statements, and valuation metrics.
- Company overview and profile
- Financial statements (income, balance sheet, cash flow)
- Key metrics and valuation ratios

### [Forex](api/forex.md)
Foreign exchange functions for currency rates and conversions.
- Exchange rates between currencies
- Currency conversion calculations

### [Cryptocurrency](api/crypto.md)
Cryptocurrency functions for digital asset prices and analysis.
- Individual crypto prices
- Major cryptocurrency data

### [Commodities](api/commodities.md)
Commodities functions for raw materials and precious metals pricing.
- Commodity price data
- Precious metals analysis

### [Economics](api/economics.md)
Economic indicators and market sentiment functions.
- US Treasury rates and yield curves
- Market sentiment indicators

### [Options](api/options.md)
Options trading functions for chains, Greeks, and implied volatility analysis.
- Options chains with Greeks
- Implied volatility analysis

### [Alpha Intelligence](api/alpha_intelligence.md)
Advanced analytics functions for risk-adjusted returns and portfolio optimization.
- Alpha and beta calculations
- Sharpe ratio analysis
- Portfolio optimization

### [Common](api/common.md)
Shared functionality including error handling, chart integration, and performance notes.
- Error handling patterns
- Chart integration guide
- Performance and dependency information
